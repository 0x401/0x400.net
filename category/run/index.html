B 站刷到一个榨干光猫所有能力的视频，解决了客厅没法同时使用千兆网络和 IPTV 的问题。

路由器为华为 TC7102，后台设置可设置绑定 IPTV 的参数，网口4 连接机顶盒用，VLAN ID 和 802.1p 的值需要去去光猫后台里查看。

批量检测网页是否被Google收录

  • 发表于：2019-01-23
  • 分类： 科学上网

工具：Google Indexation Tester
使用方法：

  1. 访问工具链接，会出现如下图提示，点击“制作副本”，然后会在你的Google表格里创建一个表格
  2. 在B列里输入你要测试的网址，然后表格右侧选择“Run”，此时就开始自动测试了。可以在“Indexed URLs”和“Non-indexed URLs”里分别查看已收录和未收录。
  3. 注意：
    1. 测试新的网址请先选“Reset”清除已存在的结果
    2. 默认支持一次测试1000个网址
    3. 下一次使用此工具直接到你Google表格里，进入Google Indexation Tester表格即可
    4. 如果工具出现错误请尝试重新“制作副本”
    5. 工具可能有时效性，注意抽查结果是否正确

详细介绍：https://www.greenlanemarketing.com/tools/google-indexation-tester/

如何自由访问宜家全球的网站

  • 发表于：2017-05-12
  • 分类： 科学上网

如果手机访问宜家移动版m.ikea.com默认是有切换国家的功能，但是电脑版www.ikea.com则不能选择国家而是直接跳转到对应的国家了，比如宜家中国www.ikea.com/cn/zh，即使你直接访问www.ikea.com/us也还是跳转到中国。

只需添加一条host指向宜家美国的ip就可以了。184.26.44.106 www.ikea.com,此时访问首页就不会跳转，而是出现选择国家的页面，完美！

Win7局域网跨网段访问共享

  • 发表于：2017-03-02
  • 分类： 科学上网

有两个网段172.16.1.x和172.16.2.x，实现互访共享。
通用的办法就是修改子网掩码，但是这里需求只需要访问某几台电脑，比如共享打印机。

首先设置防火墙规则，在目标机器上添加入站规则，UDP端口137、138，TCP端口139、445，此时用\\ip的形式就已经可以访问了。
如果要用\\计算机名访问的话，则需要在客户机的hosts文件里加一条记录，比如:172.16.1.2 PC1，然后就能通过\\PC1来访问了。

2.设置本地连接的DNS为127.0.0.1

在CMD里用nslookup命令看看是否工作在正常，如果出现请求超时，说明Acrylic的DNS有问题，这时需要修改配置文件。

Flyme3.2如何使用Mi-pop

  • 发表于：2013-12-14
  • 分类： 科学上网

FLyme 2.0 后最大的亮点莫过于 Smartbar 了，官方一直引以为豪，但也是最具争议的，我反正一点也不喜欢，太多应用会出先双底栏，解决方法就是去掉 Smartbar，我用的机锋论坛的一个去“SB”的方法。

去掉后就又一个问题，有些程序界面是没有返回或者菜单按钮的，在 Flyme 2.0 时代 我用的是 Mi-pop 这个软件，但它在 Flyme 3.0 后不能用了，因为它不适用于 Andoid 4.2，如何解决？

很简单，用“幸运破解器”这个软件，运行程序，选择工具-破解 Android 内核，然后选择前 3 项，记住不要选择红色的，重启后 Mi-pop 又能正常使用咯。
